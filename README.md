
# Recipe-Box_Rails


A user-friendly recipe-box app (**Recipe Box**), using Rails that has the following
models - Recipe, Ingredient, Directions.

![Alt text](http://savvycomsoftware.com/wp-content/uploads/2013/09/RubyOnRails1.png)

**Technologies used**
- Ruby on Rails
- HAML, simple_form
- Bootstrap theme
