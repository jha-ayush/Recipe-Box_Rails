
# Recipe-Box_Rails


A user-friendly recipe-box app (**Recipe Box**), using Rails that has the following
models - **A User has many Recipes. Each Recipe has many Ingredients & Directions.**

**Technologies used**
- Ruby on Rails
- Ruby Gems
- Bootstrap theme
